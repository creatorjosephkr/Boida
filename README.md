# 🎉 보이다(Boida) v1.0.0 — 최초 출시

아이폰 화면을 Windows PC와 Mac에서 무선으로 미러링할 수 있는 **보이다(Boida)** 첫 번째 정식 버전을 출시합니다!

AirPlay 기반으로 아이폰에 별도 앱 설치 없이, 제어 센터에서 바로 PC/Mac에 화면을 띄울 수 있습니다.

---

## 📦 다운로드

| 파일 | 플랫폼 | 설명 |
|------|--------|------|
| `Boida-v1.0.0-for-Windows-x64.exe` | Windows 10/11 | Windows 설치 파일 (권장) |
| `Boida-v1.0.0-for-Mac-universal.dmg` | macOS (통합) | Apple Silicon + Intel 통합 |
| `Boida-v1.0.0-for-Mac-arm64.dmg` | macOS Apple Silicon | M1 이상 ~ |
| `Boida-v1.0.0-for-Mac-intel-x64.dmg` | macOS Intel | 인텔 Mac 전용 |

---

## 🚀 사용 방법

### Windows
1. `Boida-v1.0.0-for-Windows-x64.exe` 설치
2. **보이다** 앱 실행
3. 아이폰 **제어 센터** → **화면 미러링** → **Boida@컴퓨터이름** 선택

### macOS
1. `Boida-v1.0.0-for-Mac-universal.dmg` 설치
2. 보이다 앱 실행
3. 아이폰 **제어 센터** → **화면 미러링** → **보이다(Boida)** 선택

> ⚠️ 아이폰과 PC/Mac이 **같은 Wi-Fi**에 연결되어 있어야 합니다.

---

## ✨ 주요 기능

- AirPlay 기반 무선 화면 미러링
- 오디오 동시 전송
- Windows: GStreamer 별도 설치 불필요 (내장)
- 저지연 H.264 디코딩
- 광고 없음, 완전 무료

---

## 📋 시스템 요구사항

- **Windows**: Windows 10 / 11 (64bit), 동일 Wi-Fi 네트워크
- **macOS**: macOS 12 Monterey 이상, GStreamer Framework, 동일 Wi-Fi 네트워크

---

## ⚠️ 알려진 제한사항

- AP 격리(Client Isolation)가 활성화된 공유기 환경에서는 동작하지 않을 수 있습니다
- Windows Defender에서 알 수 없는 게시자 경고가 표시될 수 있습니다 (정상, 무시하고 설치)
- macOS: 공증(Notarization) 미서명 앱으로 최초 실행 시 시스템 설정에서 허용 필요
